# 📱 Parul University Attendance App

**PU Attendance** is a native Android application designed to streamline and modernize the process of tracking student attendance at Parul University. This app provides a seamless interface for securely logging sessions and computing attendance metrics.

---

## 🛠️ Tech Stack & Architecture

- **Platform**: Android Native
- **Build System**: Gradle
- **Target OS**: Android

---

## 🚀 Getting Started

This repository contains an Android Studio project. Follow these steps to run the application locally or deploy it to a physical device/emulator.

### Prerequisites
1. **Android Studio** (Flamingo or later recommended).
2. Android SDK installed with platform tools.
3. An Android Virtual Device (AVD) or a physical Android tester connected via USB debugging.

### Building the App

1. Clone the repository:
   ```bash
   git clone https://github.com/Shabber10/ParulUniverstyAttendence.git
   ```

2. Open **Android Studio**.

3. Click **File > Open**, and select the `ParulUniverstyAttendence` directory you just cloned.

4. Allow Gradle to sync completely. If prompted to upgrade Gradle, accept.

5. Click the green ▶️ **Run 'app'** button in the top toolbar to deploy to your emulator or physical device.
